Practical 9

Create a user registration system using JSP and Servlet with MySQL database.

How to Run in VS Code

1. Install JDK

2. Install Apache Tomcat

3. Install MySQL Server

4. Install these VS Code Extensions:
   - Extension Pack for Java
   - Tomcat for Java

5. Download MySQL JDBC Driver

6. Create database:

CREATE DATABASE userdb;

7. Create table:

CREATE TABLE users(
    name VARCHAR(50),
    email VARCHAR(50),
    password VARCHAR(50)
);

8. Create folder structure:

project/
│
├── register.jsp
│
└── WEB-INF/
    │
    ├── web.xml
    │
    └── classes/
        │
        └── RegisterServlet.java

9. Compile Servlet:

javac -cp ".;tomcat/lib/servlet-api.jar;lib/mysql-connector-j-9.7.0.jar" ^
WEB-INF/classes/RegisterServlet.java

10. Copy project folder into:

apache-tomcat/webapps/

11. Start Tomcat Server

12. Open browser:

http://localhost:8080/projectname/register.jsp

13. Fill registration form

14. User data will store
inside MySQL database
