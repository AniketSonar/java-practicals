Practical 6

Create a client-server application using TCP sockets to echo client messages.

How to Run

1. Save the files as:
   Server.java
   Client.java

2. Open terminal in the folder

3. Compile:
javac Server.java Client.java

4. Start server first:
java Server

5. Open another terminal

6. Run client:
java Client

7. Enter messages from client

8. Server will echo messages

9. Type exit to stop
