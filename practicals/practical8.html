Practical 8

Demonstrate GET and POST methods using Servlet programming.

How to Run in VS Code

1. Install JDK

2. Install Apache Tomcat

3. Install these VS Code Extensions:
   - Extension Pack for Java
   - Tomcat for Java

4. Create folder structure:

project/
│
├── index.html
│
└── WEB-INF/
    │
    ├── web.xml
    │
    └── classes/
        │
        └── MyServlet.java

5. Compile Servlet:

javac -cp "tomcat/lib/servlet-api.jar" ^
WEB-INF/classes/MyServlet.java

6. Copy project folder inside:

apache-tomcat/webapps/

7. Start Tomcat Server

8. Open browser:

http://localhost:8080/projectname/

9. Test GET and POST forms
