Practical 7

Connect Java application to database and execute SQL queries using GUI interface.

How to Run

1. Download MySQL JDBC Driver

Website:
https://dev.mysql.com/downloads/connector/j/

2. Create folder:
lib

3. Put JDBC jar file inside lib folder

Example:
lib/mysql-connector-j-9.7.0.jar

4. Create MySQL database:
testdb

5. Save file as:
DatabaseGUI.java

6. Open terminal in project folder

7. Compile:

javac -cp ".;lib/mysql-connector-j-9.7.0.jar" DatabaseGUI.java

8. Run:

java -cp ".;lib/mysql-connector-j-9.7.0.jar" DatabaseGUI

9. Enter SQL queries in GUI

Example:
SELECT * FROM student;

10. If using XAMPP,
password may be empty:

"root", ""
